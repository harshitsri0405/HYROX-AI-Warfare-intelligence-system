# HYROX — AI Hybrid Warfare Intelligence System

AI-powered hybrid warfare intelligence and battlefield simulation platform built using Python, Streamlit, HTML, JavaScript, and Leaflet.

## Features

- Real-time battlefield simulation
- AI threat analysis
- Interactive warfare maps
- Streamlit dashboard
- Surveillance visualization

## Technologies Used

- Python
- Streamlit
- HTML/CSS/JavaScript
- Leaflet.js
- Machine Learning

## Run Project

```bash
streamlit run app.py
```
